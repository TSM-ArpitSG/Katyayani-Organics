# 📋 Assignment: Katyayani Task Management Portal

## 🎯 Objective
Build a simple **Task Management Portal** using the **MERN stack**.

## 🚀 Candidate Requirements
- **AI Tools:** Candidates should use AI tools and must:
  - Share their AI prompts.
  - Explain their code clearly.

## ✨ Core Features (Mandatory)

### 1️⃣ Add Task
- **Title** (required)
- **Description** (optional)
- **Status:** Pending / Completed
- **Created At** (auto-generated)
- *Other attributes as deemed necessary.*

### 2️⃣ View Tasks
- Display all tasks in a list.
- Show status clearly.
- Show created date.
- *Other attributes as deemed necessary.*

### 3️⃣ Mark Task as Completed
- Toggle status.
- Add filtering (**All / Pending / Completed**).

## 🛠️ Tech Constraints
- **Frontend:** React (Vite)
- **Backend:** Node.js (Express) / Next.js
- **Database:** MongoDB
- **Authentication:** User authentication required.

- **GitHub repo link**
- **README** with:
  - How to run.
  - AI prompts used.
  - What AI generated vs what was modified.
  - **API design** [NON AI GENERATED]
  - **State management** [NON AI GENERATED]
